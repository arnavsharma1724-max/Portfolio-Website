# Arnav Sharma — Portfolio Website

This is a fast, single-page portfolio website built from the details in your documents (CV/certificates).

## Run locally (Windows)

### Option A: Open directly
- Open `index.html` in your browser.

### Option B: Use a tiny local server (recommended)
From PowerShell:

```powershell
cd "c:\Users\Arnav sharma\OneDrive\Desktop\arnav-portfolio"
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Edit content

Open and edit:
- `index.html` (all text/sections)
- `styles.css` (design)
- `script.js` (theme toggle, mobile menu, contact-copy form)
